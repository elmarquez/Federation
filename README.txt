Federation Modeler: a propagation based parametric modeler.
October 10, 2006

Requirements & Usage -----------------------------------------------------------

Federation Modeler, or just Federation, is a propagation based parametric 
modeler developed to explore and test ideas about how to adapt to change in a 
design process.

Federation requires the following software be installed on your system:

    * Java 1.5.0 or above
    * Java 3D 1.51 or above

To run Federation, locate Federation.jar and execute the following from the 
command line:

    java.exe Federation.jar
